About Water Accounting App

WaterAccounting.app automatically generates a water balance report over any selected area by the user. The tool make use of remotely sensed datasets to generate various components of water balance laid over to a report in html and pdf format.

Welcome to WaterAccounting.app

Generate a report in 4 steps on water & land use assessment of your study area.

1. Add layers

Upload study area boundary by importing a GeoJSON or KML file OR select from pre-stored national boundaries and large basin boundaries OR user can digitize a boundary interactively in the map.

2. Select study area

All the layers added by the user are listed here. User can select the study area from drop down. The selected study area will be displayed in the interactive map.

3. Time Range

After selecting the study area, user can select a time range, start year and end year. Currently the report is generated only at annual scale.

4. Select products

Select the Precipitation and ET products user want to use for generating reports.The products supported are CHIRPS, GPM, PERSIANN for precipitation and SSEBop, WaPOR for ET.
